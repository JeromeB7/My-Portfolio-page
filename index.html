MOVALA AI (Miraculously Operating Virtual Assistant Like Alexa)

This Python-based AI assistant is inspired by Tony Stark's J.A.R.V.I.S. from Iron Man. It is designed to interact with users in a conversational manner, open specified applications, play songs, and browse websites on command. The assistant utilizes the Gemini API to respond to various prompts, with all interactions being stored locally for future reference. The project showcases an integration of Python programming with the Gemini API to create a multifunctional and interactive tool, mirroring the capabilities of its cinematic counterpart.
Github link: MOVALA AI

SORTING VISUALIZER

A web application to visually demonstrate various popular sorting algorithms, including Heap Sort, Bubble Sort, Insertion Sort, and Selection Sort using cylinders. The application provides auditory feedback during sorting operations using the AudioContext, webkitAudioContext, and window.webkitAudioContext APIs. Structured with HTML, styled with CSS, and made responsive with JavaScript. Sorting algorithms are implemented in JavaScript.
Github link: Sorting-Visulaizer
